# Diabetes-Prediction-System
Machine Learning-based Diabetes Prediction System using Random Forest and SMOTE for accurate early diabetes risk assessment.
## Overview

The Smart Diabetes Prediction System is a Machine Learning-based healthcare application that predicts the likelihood of diabetes using patient health parameters. The project utilizes data preprocessing, SMOTE for handling class imbalance, and a Random Forest Classifier for accurate predictions.

A user-friendly **Gradio web interface** has been integrated, allowing users to enter health-related information and receive real-time diabetes predictions instantly.

## Dataset Features

The model uses the following medical attributes:

- Pregnancies
- Glucose Level
- Blood Pressure
- Skin Thickness
- Insulin
- BMI
- Diabetes Pedigree Function
- Age
